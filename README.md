🌐 Web Development Projects
This repository contains three beginner–friendly web development projects built with HTML, CSS, and JavaScript. Each project demonstrates core front‑end skills such as layout, styling, responsiveness, and interactivity.

📌 Project 1: Landing Page
Overview
A visually polished static landing page for a product, service, or brand. Designed to practice foundational HTML/CSS layout skills.

Features
Sticky navigation bar with 3+ links

Hero section with headline, subheadline, and CTA button

Multiple content sections (Features, About, Testimonials)

Footer with placeholder contact/social links

Responsive layout using Flexbox/Grid

Consistent color palette and typography

How to Run
Open index.html in your browser.

Customize style.css for branding.

No JavaScript required for this project.

📌 Project 2: Temperature Converter
Overview
An interactive website that converts temperatures between Celsius, Fahrenheit, and Kelvin using JavaScript.

Features
Input validation for numeric values

Conversion formulas implemented in JS

Real‑time conversion output

Responsive design for desktop and mobile

How to Run
Open index.html in your browser.

Ensure script.js is linked properly.

Enter a temperature value and select conversion type.

📌 Project 3: Personal Portfolio
Overview
A digital résumé website showcasing skills, projects, and contact information.

Features
Hero section with name, role, and avatar

About Me section with background and interests

Skills section (grid/list format)

Projects section with placeholder cards and GitHub links

Contact section with email + social icons

Smooth scroll navigation

Responsive design with consistent branding

Optional JavaScript enhancements (smooth scroll, animations, dark mode toggle)

How to Run
Open index.html in your browser.

Customize style.css for your personal branding.

Enhance with script.js for interactivity.

🚀 Getting Started
Clone the repository and open each project folder:

bash
git clone https://github.com/yourusername/web-projects.git
cd web-projects
Each project has its own index.html, style.css, and optional script.js.
